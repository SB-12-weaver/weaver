프로젝트 소개

기업의 핵심 자산인 인적 자원 데이터를 안전하게 저장하고, 대량의 데이터를 주기적으로 백업 및 관리하는 서비스 구축
프로젝트 기간: 2026.05.14 ~ 2026.05.26

기술 스택
- Framework: Spring Boot
- Database & ORM : PostgreSQL, Spring Data JPA
- Documentation : springdoc-openapi (Swagger)
- Scheduling : Spring Scheduler
- Utility : MapStruct
- 협업 : Git & GitHub, notion
